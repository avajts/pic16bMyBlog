Preparing the Dataframe

Next I created a function to transform a wide formated dataframe into a long formated dataframe. This definition also cleans up the Month and Temp columns and drops and NaN values from the Temp column.

def prepare_df(df):
    """
    prepares a piece of wide format dataframe into a long format data frame
    """
    # melt to the long format table
    df = df.melt(
        id_vars = ["ID", "Year"],
        value_vars = [f"VALUE{i}" for i in range(1, 13)],
        var_name = "Month",
        value_name = "Temp"
    )

    # cleaning month and temp
    df["Month"] = df["Month"].str[5:].astype(int)
    df["Temp"]  = df["Temp"] / 100
    df = df[~np.isnan(df["Temp"])]

    return df

2. Query Function

Importing Functions

In a separate file called ‘climate_database.py’, I wrote multiple functions which returns the data wrangling and visualizations seen in this blog post. The first one to be imported is ‘query_climate_databse’:

from climate_database import query_climate_database
import inspect
print(inspect.getsource(query_climate_database))
def query_climate_database(db_file, country, year_begin, year_end, month):
    """
    Arguments:
        db_file = the file name for the database
        country = a string giving the name of a country 
        year_begin = int giving the earliest year (inclusive)
        year_end = int giving the latest year (inclusive)
        month = int giving the month of the year
    Return Value:
        The function returns a pandas dataframe which contains the temperature readings for the specified country, 
        in the specified date range, in the specified month of the year. The dataframe will have the following columns: 
        NAME = The station's name
        LATITUDE = The latitude of the station 
        LONGITUDE = The longitude of the station 
        Country = The name of the country in which ithe station is located
        Year = The year in which the reading was taken
        Month = The month in which the reading was taken
        Temp = The average temperature at the specified station during the specified year and month
    """

    cmd = \
        f"""
        SELECT S.NAME, S.LATITUDE, S.LONGITUDE, C."Name" AS Country, T.Year, T.Month, T.Temp
        FROM temperatures T
        JOIN stations S ON T.ID = S.ID
        JOIN countries C ON substr(S.ID, 1, 2) = C."FIPS 10-4"
        WHERE C.Name = '{country}'
        AND T.Year BETWEEN {year_begin} AND {year_end}
        AND T.Month = {month}
        ORDER BY S.NAME ASC, T.Year ASC;
        """
    
    with sqlite3.connect(f"{db_file}") as conn:
        df = pd.read_sql_query(cmd, conn)
        
    return df

3. Geographic Scatter Function for Yearly Temperature Increase

Big Question

We were asked to create a visualization the addresses the following question: > How does the average yearly change in temperature vary within a givin country? The best way to answer this would be to create a visualization for the given country, showing this yearly change in temperature.

Importing Functions

In ‘temperature_coefficient_plot’, I create a dataframe using our previously defined function, ‘query_climate_database’ on the given input parameters. Next, I group the dataframe by station, then filters out the stations that don’t have enough observations.

I created a function ‘compute_trend’ within this function, which computes the temperature trend (slope) per station. It uses linear regression’s model coefficient from sklearn’s library as the return value for the slope. We group the filtered dataframe by name, latitude, and longitude, then apply the ‘compute_trend’ function.

Last is to create the figure, which is done using plotly’s scatter_mapbox function. I added a colorbar to the figure and updated the margin for formatting.

import plotly.express as px
from sklearn.linear_model import LinearRegression

from climate_database import temperature_coefficient_plot
print(inspect.getsource(temperature_coefficient_plot))
def temperature_coefficient_plot(db_file, country, year_begin, year_end, month, min_obs=10, **kwargs):
    """
    Arguments:
        db_file = the file name for the database
        country = a string giving the name of a country 
        year_begin = int giving the earliest year (inclusive)
        year_end = int giving the latest year (inclusive)
        month = int giving the month of the year
    Return Value:
        This function returns an interactive geographic scatterplot with a point for each station. 
        The color for each point reflects the yearly change in temperature during the specified month and time period at that station.
    """

    # Get our specific df given input params using our previously defined function
    df = query_climate_database(db_file, country, year_begin, year_end, month)

    # Group by station and filter stations with sufficient observations
    station_counts = df.groupby("NAME")["Year"].nunique()
    valid_stations = station_counts[station_counts >= min_obs].index
    df_filtered = df[df["NAME"].isin(valid_stations)]

    # Function to compute temperature trend (slope) per station
    def compute_trend(group):
        if len(group) < min_obs:
            return np.nan  # Skip stations with insufficient data

        X = group["Year"].values.reshape(-1, 1) 
        y = group["Temp"].values

        model = LinearRegression()
        model.fit(X, y)
        return model.coef_[0]  # Extract slope

    # Compute slope per station
    trends = df_filtered.groupby(["NAME", "LATITUDE", "LONGITUDE"]).apply(compute_trend).reset_index(name="Trend")

    # Drop stations with NaN trends
    trends = trends.dropna()
    
    fig = px.scatter_mapbox(trends,
                            lat = "LATITUDE",
                            lon = "LONGITUDE",
                            hover_name = "NAME",
                            color = "Trend",
                            range_color = [-0.1, 0.1], 
                            zoom = 2,
                            height = 300,
                            hover_data = {"Trend": ":.4f"}, # SHpws trend rounded to 2 decimals
                            color_continuous_scale = kwargs.get("color_continuous_scale", px.colors.diverging.RdGy_r),
                            mapbox_style = kwargs.get("mapbox_style", "carto-positron"),
                            title = f"Estimates of Yearly Increase in Temperature in Month {month} for Stations in {country}, years {year_begin}-{year_end}"
                           )

    # Center colorbar at 0 
    fig.update_layout(
        coloraxis_colorbar = dict(
            title = "Estimated Yearly Increase (°C)", 
            tickvals=[-0.1, -0.05, 0, 0.05, 0.1],  # Custom tick values
            ticktext=["-0.1", "-0.05", "0", "0.05", "0.1"]  # Custom tick labels
        ))
    
    fig.update_layout(margin={"r":50,"t":50,"l":50,"b":0})
    
    return fig

4. Two More Visualizations

First Visualization

After thinking on our big question more, I thought of two new ways we could visualize temperature change yearly in a given country. The first is with a line plot showing the average temperature over time in a country for the given time interval. When looking at this visualization, we can compare warming/cooling trends in different parts of the country.

Importing Functions

First, I created ‘query_avg_temp_over_time’, which creates a database similar to ‘query_climate_database’ but instead only returns name, latitude, and the average temperature for each year.

from climate_database import query_avg_temp_over_time
print(inspect.getsource(query_avg_temp_over_time))
def query_avg_temp_over_time(db_file, country, year_begin, year_end, month):
    """
    Returns the average temperature for each year in a given country as a dataframe.
    """
    cmd = f"""
        SELECT T.Year, S.LATITUDE, AVG(T.Temp) AS AvgTemp
        FROM temperatures T
        JOIN stations S ON T.ID = S.ID
        JOIN countries C ON substr(S.ID, 1, 2) = C."FIPS 10-4"
        WHERE C.Name = '{country}'
        AND T.Year BETWEEN {year_begin} AND {year_end}
        AND T.Month = {month}
        GROUP BY T.Year, S.LATITUDE
        ORDER BY T.Year ASC;
    """
    with sqlite3.connect(f"{db_file}") as conn:
        df = pd.read_sql_query(cmd, conn)
        
    return df

Next, I created ‘plot_avg_temp_over_time’ which creates a line plot with two facets for the North and South region of the given country. First, I creat a dataframe using ‘query_avg_temp_over_time’, then define a region column in the dataframe based on the latitude’s median. Last is to use plot;y’s line function to create our visualization.

from climate_database import plot_avg_temp_over_time
print(inspect.getsource(plot_avg_temp_over_time))
def plot_avg_temp_over_time(db_file, country, year_begin, year_end, month):
    df = query_avg_temp_over_time(db_file, country, year_begin, year_end, month)

    # Define "north" and "south" based on latitude
    df["Region"] = df["LATITUDE"].apply(lambda x: "North" if x >= df["LATITUDE"].median() else "South")

    # Get the yearly data for long term changes instead of seasonal
    df_yearly = df.groupby(["Year", "Region"])["AvgTemp"].mean().reset_index()

    fig = px.line(df_yearly, 
                  x = "Year", 
                  y = "AvgTemp", 
                  color = "Region", 
                  title = f"Average Temperature Over Time in {country} ({year_begin}-{year_end})", 
                  labels = {"AvgTemp": "Average Temperature (°C)"},
                  facet_col = "Region",
                  facet_col_spacing = 0.15
                 )

    fig.update_layout(showlegend=False)
    return fig

Second Visualization

The other way I thought of visualizing temperature change yearly in a given country is by which weather stations recorded the most extreme temperatures. This can be a bar chart showing the stations with the highest and lowest recording temperatures. I believe this is interesting because it can help us identify the most extreme weather situations, which can be useful for things like climate impact studies.

Importing Functions

First, I created ‘query_extreme_temps’ which creates a dataframe containing the weather stations with the highest and lowest recorded temperatures using SQL. The dataframe only returns with columns name, MinTemp, and MaxTemp.

from climate_database import query_extreme_temps
print(inspect.getsource(query_extreme_temps))
def query_extreme_temps(db_file, country, year_begin, year_end):
    """
    Gets the weather stations with the highest and lowest recorded temperatures.
    """
    cmd = f"""
        SELECT S.NAME, MIN(T.Temp) AS MinTemp, MAX(T.Temp) AS MaxTemp
        FROM temperatures T
        JOIN stations S ON T.ID = S.ID
        JOIN countries C ON substr(S.ID, 1, 2) = C."FIPS 10-4"
        WHERE C.Name = '{country}'
        AND T.Year BETWEEN {year_begin} AND {year_end}
        GROUP BY S.NAME
        ORDER BY MaxTemp DESC;
    """
    with sqlite3.connect(f"{db_file}") as conn:
        df = pd.read_sql_query(cmd, conn)
        
    return df

Next, I created ‘plot_extreme_temps’ which creates a boxplot showing the stations with the highest and lowest temperatures. I define a dataframe using ‘query_extreme_temps’, then using the top_n stations parameter find that many largest and smallest maximum and minimum temperatures from the dataframe. Then I use .melt() to combine these top_n maximum and minimum temperatures and reshape the data. Last is to use plotly’s bar function to create the bar chart with our melted dataframe.
